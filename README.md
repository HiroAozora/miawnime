MiawNime is a modern, ad-free anime streaming web application built with **Next.js 16**. Designed for speed, comfort, and a premium user experience.

## ‚ú® Features

- **Ad-Free Experience**: No intrusive popups or gambling ads. Just anime.
- **Modern UI**: Dark mode by default, glassmorphism design, and responsive layout.
- **Release Schedule**: Real-time anime release schedule grouped by day.
- **Watch History**: Automatically saves your last watched episode and position.
- **Search**: Fast anime search functionality.
- **Optimization**: Built on Next.js App Router with server-side caching (ISR) for optimal performance.

## üöÄ Getting Started

1.  **Clone the repository**

    ```bash
    git clone https://github.com/HiroAozora/miawnime.git
    ```
